# Faborite VS Code Extension

A Visual Studio Code extension for managing Faborite syncs directly from your editor.

## Features (Planned)

- Browse workspace lakehouses in sidebar
- Quick sync commands from command palette
- Visual config editor
- Sync progress notifications
- IntelliSense for synced table names in notebooks
